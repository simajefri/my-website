# Dr. Sima Jafarikhah - Academic Website

A professional academic website showcasing research, teaching, and personal interests with a modern, responsive design.

## Features

- **Professional Design**: Clean, modern interface with warmer tones
- **Responsive Layout**: Mobile-first design that works on all devices
- **Smooth Navigation**: Easy-to-use navigation with smooth scrolling
- **Interactive Elements**: Hover effects, animations, and smooth transitions
- **Accessibility**: WCAG compliant with keyboard navigation support

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # CSS styles and responsive design
├── script.js           # JavaScript functionality
├── pic.JPG            # Profile photo
├── website.md          # Website specification
└── README.md           # This file
```

## Getting Started

1. **Open the website**: Simply open `index.html` in your web browser
2. **Local development**: Use a local server for best experience
3. **Customization**: Modify the HTML, CSS, and JavaScript files as needed

## Customization

### Profile Photo
- Replace `pic.JPG` with your own photo
- Recommended size: 350x350 pixels or larger
- The photo will be automatically cropped to a circle

### Content Updates
- **Personal Information**: Update name, title, and contact details in `index.html`
- **Research Areas**: Modify research descriptions and areas of expertise
- **Teaching**: Update course information and descriptions
- **Personal Interests**: Customize hobbies and personal activities

### Styling
- **Colors**: Modify the CSS variables in `styles.css`
- **Fonts**: Change the Google Fonts import in `index.html`
- **Layout**: Adjust grid layouts and spacing in `styles.css`

### LinkedIn Integration
- Update the LinkedIn URL in the hero section buttons
- Modify the footer LinkedIn link
